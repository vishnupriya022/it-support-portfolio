# IT Support Ticketing Projects Portfolio

Here are my 4 IT Support Ticketing projects:

1. [Printer Not Working – IT Support](https://github.com/vishnupriya022/printer-not-working-it-support)  
   - A simulated IT support helpdesk project documenting the troubleshooting and resolution of a printer not detected issue.

2. [Outlook Not Working – IT Support](https://github.com/vishnupriya022/outlook-not-working-it-support)  
   - A project focused on diagnosing and documenting solutions to Outlook/email not working problems.

3. [User Account Lockout – IT Support](https://github.com/vishnupriya022/user-account-lockout-it-support)  
   - A project managing issues related to user account lockouts and helpdesk resolution steps.

4. [Internet Connectivity Issue – IT Support](https://github.com/vishnupriya022/internet-connectivity-issue-it-support)  
   - A project for tracking and resolving internet/network connectivity problems.
  
5. [Password Reset – IT Support](https://github.com/vishnupriya022/password-reset-it-support)
 
   - This project demonstrates how an IT support team handles password reset requests, including user verification, account status checks, password reset procedures, and ticket closure steps.

6. [VPN Connectivity Issue – IT Support](https://github.com/vishnupriya022/vpn-connectivity-issue–it-Support)
 
   - This project demonstrates a real-world IT support scenario where a user is unable to connect to a VPN. It shows step-by-step troubleshooting, resolution, and ticket closure — fully documented for GitHub portfolio use.
